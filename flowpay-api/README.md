# FlowPay API

API para gerenciar serviços de atendimento.

## O que precisa ter instalado

- Node.js (versão 18 ou superior)
- npm

## Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Inicie a aplicação:
```bash
npm run start:dev
```

A aplicação vai rodar na porta 3000.

A documentação da API está disponível em: http://localhost:3000/api/docs

Caso queira, tem um mini desenho da aplicação na pasta arch.

# FlowPay Frontend

Sistema de central de relacionamento da FlowPay.

## O que precisa ter instalado

- Node.js (versão 18 ou superior)
- npm

## Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Certifique-se de que a API está rodando em `http://localhost:3000`

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Acesse `http://localhost:3001` no navegador

## Build para produção

```bash
npm run build
```
